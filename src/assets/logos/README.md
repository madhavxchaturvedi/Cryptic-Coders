# Logo Assets

Please add your Cryptic Coders club logo to this directory:

## Required File:
**club-logo.png** - Cryptic Coders club logo
- Your club's official logo
- Recommended size: 200x200px or higher
- Transparent background (PNG) preferred for best results
- Will be displayed in the navbar and footer

## How to add your logo:
1. Save your logo image to this folder (`/src/assets/logos/`)
2. Name it exactly: `club-logo.png`
3. The website will automatically display it

If the logo file is not found, a fallback "</>" symbol will be shown.
